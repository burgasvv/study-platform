
# ___Study-Platform API___

## Версии
+ Ktor 3.4.0
+ OpenJDK 25
+ Gradle 9.4.1

## Основной стек зависимостей
+ Exposed DB
+ Postgresql
+ Authentication Basic
+ Routing
+ Serialization
+ Redis

## Описание
    Server API платформа обучения для обучения на курсах и выполнения проектов студентами

## Запуск приложения
* > Запустить приложение из среды разработки или поднять с помощью docker-compose.yaml
* > Запустить запросы с помщью request-test.http или postman
